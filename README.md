# AVersion
Drawings and tables

**Подключение проводов - последовательное:**
***сегмент 1***
- (1 устройство)(контакт N)(2 часть контакта) - (1-начало)(сегмент 1)
- (2 устройство)(контакт M)(1 часть контакта) - (2-конец)(сегмент 1)
***сегмент 2***
- (2 устройство)(контакт M)(2 часть контакта) - (1-начало)(сегмент 2)
- (3 устройство)(контакт K)(1 часть контакта) - (2-конец)(сегмент 2)
***сегмент 3***
- (3 устройство)(контакт K)(2 часть контакта) - (1-начало)(сегмент 3)

*соединения 2 конца сегмента провода нет если сегмент провода выведен вовне*

Хотя совсем не обязательно начало сегмента провода втыкается а 1 часть контакта (может как попало - и во 2), нумерация сегментов хорошо бы соответствовала последовательности соединений (для компактности и понятности выводимого отчета).
